⚙️ Configuração Google Sheets

1
Acesse console.cloud.google.com → crie um projeto (ex: financas-vc).
2
Vá em APIs & Services → Library e ative:
Google Sheets API e Google Drive API.
3
Vá em APIs & Services → OAuth consent screen → escolha External → preencha o nome do app → salve.
4
Vá em Credentials → Create Credentials → OAuth Client ID. Tipo: Web application.
Em Authorized JavaScript Origins adicione a URL onde este arquivo está (ex: http://localhost ou https://seusite.com).
5
Copie o Client ID gerado e cole abaixo. Salve e clique em Entrar com Google no topo.
